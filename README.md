
## DESCRITPION 

SharpAlgos is an collection of Algorithms and Data Structures written in C# 11

Your contribution is welcome.
 
#### Requirements:
- [Visual Studio 2022](https://visualstudio.microsoft.com/downloads/)
- .Net Core 7
